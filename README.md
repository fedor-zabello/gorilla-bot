# Gorilla HC Telegram Bot

A Telegram bot for fans of the best hockey team from Saint-Petersburg, Gorilla HC.

## Features
*   **Match Schedule:** Get information about the team's upcoming matches.
*   **Latest Results:** Check the score and details of the most recent game.
*   **Automatic Notifications:** All subscribers receive reminders about upcoming matches and immediate updates with results once a game ends.

## How to Use
1.  Find the bot on Telegram: @gorilla_hc_bot
2.  Send the `/start` command to begin.
3.  Use the menu or commands like `/upcoming` and `/last_result` to get information.
